I'm Victor, an `Electrical and Electronics Engineer` with a speciality in embedded and real-time systems programming.

I'm particularly interested in writing safe and concise code for microcontrollers with a specific functionality that meets given required specifications. I prefer using the C programming language in programming microcontrollers as it allows direct hardware access and has a significantly faster code execution compared to other programming languages.

This personal blog is an archive of topics and projects in embedded system programming that I find interesting. Therefore, this blog acts as a reference to what I document and is available to be consumed by all. Welcome!!
